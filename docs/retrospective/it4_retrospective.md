In this iteration, we added most of the features that will be in the final version of the app. These features were:
- Add ability to be the driver for a ride
- Have a list of riders who will be associated with a ride
- Allow the poster to approve/deny ride requests for that ride
- Add tabs for friends list, messages, profile
- Add groupchat functionality for the ride

On the positive side, we made the most progress on this iteration. We were able to quickly add many features that are crucial for the practical use of the app. From here, all we have to do is add the friends feature, and make a few data structure updates and UI polishes. 

Things that can be improved going forward are better communication of work progress, and finishing work earlier so we have more time to test. We were able to test in the development environment, but there were a few issues with the deployed app that were solved on Monday.
