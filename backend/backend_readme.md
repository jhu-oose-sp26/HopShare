1. install mongodb at this [link](https://www.mongodb.com/try/download/community) and install Compass when prompted
2. start a server at the default localhost using Compass
3. make an .env file based off of .env.example to contain all environment variables
4. check out the package.json to see the pnpm commands

Note: if your MongoDB Compass isn't updating properly refresh it using Ctrl+R or Cmd+R
